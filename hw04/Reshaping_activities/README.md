## Repo for Reshaping activites in 4th homework stat545 

This is the *readme file* for the reshaping activities in fourth homework. 
Different reshaping functions such as **melt()**, **dcast()**, **gather()**, **spread()**, **expand** etc that belongs to **tidyr** package was explored in this assignment.**Knitr** and **kable** package was used for making nice tables.

You can find the [activity 1 markdown file](https://github.com/abishekarun/STAT545-hw-rajendran-arun/blob/master/hw04/Reshaping_activities/activity1.md) and the [activity 5 markdown file](https://github.com/abishekarun/STAT545-hw-rajendran-arun/blob/master/hw04/Reshaping_activities/activity5.md)
